a simple text editor with some neat features on the side

View project on GitHub

What is this project?

A text editor.

Why?

So programmers can edit text.

I wanted to create a text editor with a "clean, distraction-free interface". There are several neat features that are included too:

  • GitHub flavored Markdown Preview

  • Material Design Hex Color Palette

  • Save Commands and Easily Run Them

  • And More (check out the wiki)

Is this project done?

I would say it's in a beta stage. It is usable and if a bug pops (which is very rare), it will not affect your work. Unless you forget to save... then that's your problem.

I want to download and run this. How?

At the moment, there are working builds for the following systems:

Windows 7+ (32-bit, 64-bit)

Available but not officially built:

Mac OS 10.9+

Ubuntu 12.04+

Fedora 21

Debian 8
